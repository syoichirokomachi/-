# 昼休みシフト最適化ツール

このアプリは、スタッフの勤務時間と休憩時間を視覚化し、最適な休憩スケジュールを作成するための **Streamlitアプリ**です。

## 機能
- CSVファイルからスタッフ情報を読み込み
- 勤務時間・休憩時間をタイムラインで表示（勤務：緑、休憩：グレー）
- 手動で休憩時間を入力可能（「:」入力対応）
- 未入力の場合は自動割り当て（9:00～17:00）
- 横軸は8:00～18:00固定で時間ラベル表示
- Excelファイルとしてスケジュールをダウンロード可能
